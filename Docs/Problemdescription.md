## Problem Description

Licensing is very very hard and a black area to most of the younger generations of developers they don't get the historical drama behind some events where some projects were forked or abandoned because of adopting a new license that changes the terms or why the companies love to make open source API components that can circumvent certain licenses rules

this project aims to make the developer understand these events what is copyleft what is permissive why choose A and not B or C and why maybe D is worse than A for the case at hand

(We are not getting into the events and practices searching on the topic is the entire point of this project which is "Awareness")
