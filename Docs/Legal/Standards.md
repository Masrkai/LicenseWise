# Standards and Organizations

Organizations and standards that define and approve software licenses.

## Organizations

| Organization | Full Name | What It Does | License Lists |
|--------------|-----------|--------------|---------------|
| **OSI** | Open Source Initiative | Maintains the Open Source Definition and approves licenses that meet it. | OSI-approved licenses |
| **FSF** | Free Software Foundation | Promotes free software and maintains a list of licenses it considers "free." | FSF-free licenses |
| **SPDX** | Software Package Data Exchange | A standard format for identifying and communicating license information. | SPDX license identifiers |
| **EUPL** | European Union Public Licence | Specifically designed for EU law and government projects. | EUPL-1.2 |

## Definitions

| Term | Plain Explanation | Who Maintains? |
|------|-------------------|----------------|
| **Open Source Definition** | The set of criteria that a license must meet to be considered "open source." | OSI |
| **Free Software Definition** | The four essential freedoms: run, study/modify, redistribute, and distribute modified versions. | FSF |
| **OSI-Approved** | A license that has been reviewed and approved by the Open Source Initiative as meeting the Open Source Definition. | OSI |
| **FSF-Free** | A license that the Free Software Foundation recognizes as providing the four essential freedoms of free software. | FSF |
| **SPDX License Identifier** | A short, standardized string (like "MIT", "GPL-2.0") used to identify licenses in machine-readable formats. | SPDX/ISO |

## Why Standards Matter

| Standard | Why It Matters |
|----------|----------------|
| **SPDX** | Machine-readable license identification. Used in package managers, CI/CD, and compliance tools. |
| **OSI Approval** | Community trust. OSI-approved licenses are widely accepted and understood. |
| **FSF-Free** | Ensures software respects user freedoms. Important for GNU/Linux distributions. |
| **EUPL** | Legal validity across EU jurisdictions. Important for government and public sector projects. |
